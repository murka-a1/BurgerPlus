# BurgerPlus
В данном проекте реализована авторизация пользователя под двумя ролями: "Администратор" и "Пользователя". После запуска приложения нужно ввести пароль и логин (будут представлены ниже), затем нужно правильно ввести капчу, ведь без нее не будет кнопки "Войти". Если вы успешно все сделали то появится окно с уведовлением "Здравствуйте администратор/пользователь, ФИО", но если была совершена ошибка при вводе пароля или логина, то появится уведомление с ошибкой. После успешного прохождения авторизации, админстратор может увидеть список всех пользователей, так же может редактировать, удаля и добавлять их. Если вы зашли под ролью пользователь, то будет просто пустое окно. 
Были использованные технологии как Visio, Visual studio, My SQL, Word, Execl.
